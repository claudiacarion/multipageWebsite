HTML CSS Artwork

In 2009 at Griffith University, I took a course called Writing for the Web. For this assignment, we were tasked to create a piece of artwork using only HTML and CSS. Using a table and coloring each cell, I created a portrait of my favorite singer. Simple, but I did end up getting an HD, which is the highest grade possible!
